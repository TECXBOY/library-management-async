# Library Management System – Async API

A Python-based asynchronous library management system built for the Object-Oriented Programming assignment at Limkokwing University of Creative Technology.

## Description

This project simulates a library system where multiple users can borrow and return books concurrently using Python's `asyncio` library. It demonstrates asynchronous programming concepts and API design principles.

## Features

- Search for books by title, author, or category
- Borrow a book (with availability check)
- Return a borrowed book
- View overdue books and calculate fines

## Technologies Used

- Python 3
- asyncio

## How to Run

```bash
python asycino.py
```
